# Workflow: 콘텐츠 제작 (교육자료 · PPT · 블로그/유튜브 보조편집)

## INPUT
- 고객사/주제 브리프 (텍스트 또는 회의록)
- 기존 참고자료 (PDF, 슬라이드, 링크)
- 톤앤매너 가이드 (있으면)

## PROCESS

### A. 고객 맞춤형 교육 설계
1. 브리프에서 학습 목표 3~5개 추출 → 사람에게 확인
2. 목표별 커리큘럼 목차 초안 작성
3. 목차 승인 후에만 상세 콘텐츠 작성 진입 (목차 단계에서 되돌리는 게 훨씬 저렴함)

### B. 강의자료(PPT) 자동 생성
1. 승인된 목차를 슬라이드 단위로 분해 (슬라이드당 핵심 메시지 1개 원칙)
2. `/mnt/skills/public/pptx/SKILL.md` 를 반드시 먼저 읽고 생성
3. 스타일: 회사 브랜드 가이드가 있으면 적용, 없으면 심플/가독성 우선

### C. 블로그 글
1. 목차 → 섹션별 핵심 문장 1개씩 먼저 작성 (뼈대)
2. 뼈대 승인 후 살 붙이기
3. 저작권: 인용은 짧게, 출처 명시

### D. 유튜브 보조 편집
1. 스크립트/자막 정리, 챕터 타임스탬프 초안
2. 실제 영상 편집(컷 편집)은 이 시스템 범위 밖 — 편집 지시서만 산출

## VALIDATE
- [ ] 학습목표/목차가 브리프 요구사항을 100% 커버하는가
- [ ] 슬라이드 1장 = 메시지 1개 원칙 지켜졌는가
- [ ] 브랜드 톤앤매너 위반 없는가
- [ ] 저작권(인용/이미지) 문제 없는가
